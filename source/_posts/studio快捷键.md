1. 查看类的结构 View -> Tool Windows -> Structure ,  快捷键
    Windows：Alt + 7
    Mac　　 ：Cmd + 7
2. 在编辑类的时候使用热键查看类的结构：(倒数第二个菜单 File Structure)
    Windows：Ctrl + F12 (默认未启用)
    Mac　　 ： Cmd + F12
3. 查看类的继承关系
    Windows：Ctrl + H
    Mac　　 ：Control + H
4. 快捷键即可自动生成布局文件 alt + enter
