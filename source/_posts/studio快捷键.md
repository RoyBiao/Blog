1. 查看类的结构 View -> Tool Windows -> Structure ,  快捷键
    Windows：Alt + 7
    Mac 　 ：Cmd + 7
2. 查看类的继承关系
    Windows：Ctrl + H
    Mac　　：Control + H
3. 快捷键即可自动生成布局文件 alt + enter 

4. 从现有代码提取新的方法
   Windows: Ctrl + Alt + M
   Mac    : Cmd + Option + M
